# OOs.Common

A family of reusable .NET libraries providing common utilities, extensions, and helpers.

## Packages

| Package | Description |
|--------|-------------|
| `OOs.Common` | Core shared utilities and types used across OOs libraries. |
| `OOs.Common.CommandLine` | Command-line argument parsing support and helpers. |
| `OOs.Common.CommandLine.Generators` | Source generator for command-line argument parsing. |
| `OOs.Common.IO.Pipelines` | Helpers and extensions for `System.IO.Pipelines`. |
| `OOs.Common.Memory` | Memory pooling and segment utilities. |
| `OOs.Common.Net` | Networking helpers and extensions. |
| `OOs.Common.Net.Http` | HTTP helpers, JWT utilities, and Web Push support. |
| `OOs.Common.Resilience` | Resilience patterns (retry, repeat, etc.). |
| `OOs.Common.Security` | Security utilities (hashing, encoding, crypto helpers). |
| `OOs.Common.Threading` | Threading and concurrency utilities. |
| `OOs.Extensions.Configuration` | Extensions for `Microsoft.Extensions.Configuration`. |
| `OOs.Extensions.Diagnostics` | Extensions for `Microsoft.Extensions.Diagnostics`. |
| `OOs.Extensions.Hosting` | Extensions for `Microsoft.Extensions.Hosting`. |

## Packaging

Packages are built using the SDK-style project system.
